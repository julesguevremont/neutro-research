## Theoretical Foundation

### Why Sleep Matters

Every complex organism sleeps. Evolution preserved this behavior across hundreds of millions of years despite its obvious costs (vulnerability, lost time). Why?

Research shows sleep is essential for:
- **Memory consolidation**: Transferring learning to long-term storage
- **Synaptic homeostasis**: Pruning unnecessary connections
- **Creative insight**: Finding associations logic wouldn't find
- **Emotional regulation**: Processing affective experiences

If sleep is fundamental to biological intelligence, it may be fundamental to *any* intelligence.

### NEXTUP Theory

NEUTRO's dream system implements the NEXTUP model (Stickgold & Zadra, 2021):

> **N**etwork **E**xploration to **U**nderstand **P**ossibilities through **T**ranscendence

Key insight: Dreams explore *weak* associationsâ€”connections the waking brain would never consider. This produces creative leaps that pure logic cannot achieve.

## What Happens in Each Phase

### TRANSITION
- Current state saved to checkpoints
- Active processes wound down
- Memory buffers prepared for replay
- Emotional state noted for processing

### SPINDLE
- Short bursts of procedural memory activation
- Skills practiced through internal simulation
- Action sequences reinforced
- Pattern connections strengthened

### DEEP
- Sequential replay of recent episodic memories
- Fact extraction and abstraction
- Transfer to semantic memory
- Pruning of weak/unused connections
- Synaptic homeostasis (reset to baseline)

### REM
- NEXTUP exploration of weak associations
- Combine recent + remote memories
- Test novel connections
- Emotional processing (reduce charge)
- Scenario simulation (rehearsal)
- Narrative integration (identity coherence)
